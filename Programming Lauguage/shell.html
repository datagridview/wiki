变量

声明变量

等于号前后不要加空格，会被认为是一条指令

Variable="Some string"
Variable=$anotherVariable

使用变量

赋值、export使用变量时，前面不加$

单引号不会展开变量

echo $Variable
echo "$Variable"
echo '$Variable'

FILE="Favorite Things.txt"
cat $FILE   ### 尝试输出两个文件: `Favorite` 和 `Things.txt`
cat "$FILE" ### 输出一个文件: `Favorite Things.txt`

readonly rword
# rword="bye"  # 如果放开注释，执行时会报错

unset dword    # 删除变量

两种引号拼接字符串

# 使用单引号拼接
name1='white'
str1='hello, '${name1}''
str2='hello, ${name1}'
echo ${str1}_${str2}
# Output:
# hello, white_hello, ${name1}

# 使用双引号拼接
name2="black"
str3="hello, "${name2}""
str4="hello, ${name2}"
echo ${str3}_${str4}
# Output:
# hello, black_hello, black

截取变量

# 返回变量的前7个字符
Length=7
echo ${Variable:0:Length}

变量的默认值

当变量为null或者""时，返回默认值，对于变量的值没有影响

echo ${Foo:-"Default"}

数组变量

数组下标从 0 开始，下标可以是整数或算术表达式，其值应大于或等于 0。

colors=(red yellow "dark blue")

echo ${colors[*]}
# Output: red yellow dark blue 
# 其中dark blue其实分离了

echo ${colors[@]}
# Output: red yellow dark blue
# 其中dark blue没有分离 还是一个整体参数

结构

if结构

if [ $Name -ne $USER ]
then
    echo "False"
else
    echo "True"
fi

case结构

# 命令块之间要用;;分隔。
case "$Variable" in
    # 列出需要匹配的字符串
    0) echo "There is a zero.";;
    1) echo "There is a one.";;
    *) echo "It is not null.";;
esac

for结构

遍历参数序列

# 循环遍历给定的参数序列:
# 变量$Variable 的值会被打印 3 次。
# 循环的位置也可以是：
# 其他命令的输出$(ls)
# 文件 file1 file2 
for Variable in {1..3}
do
    echo "$Variable"
done

经典for循环

for ((a=1; a <= 3; a++))
do
    echo $a
done

while 结构

while [ true ]
do
    echo "loop body here..."
    break
done

函数结构

# 经典

{
    echo "Arguments work just like script arguments: $@"
    echo "And: $1 $2..."
    echo "This is a function"
    return 0
}

# 简单
bar ()
{
    echo "Another way to declare functions!"
    return 0
}

# 调用
foo "My name is" $Name

简单命令选择

echo "Always executed" || echo "Only executed if first command fails"
echo "Always executed" && echo "Only executed if first command does NOT fail"

# 在if语句中需要用[  ]包裹起每个条件
if [ $Name == "Steve" ] && [ $Age -eq 15 ]
then
    echo "This will run if $Name is Steve AND $Age is 15."
fi

表达式

echo $(( 10 + 5 ))
# 在算数表达式中，使用变量无需带上$前缀

# 一个指令可用 $( ) 嵌套在另一个指令内部：
# 以下的指令会打印当前目录下的目录和文件总数
echo "There are $(ls | wc -l) items here."

# 反引号 `` 起相同作用，但不允许嵌套
# 优先使用 $(  ).
echo "There are `ls | wc -l` items here."

# 计算表达式的值
expr $a / $b

运算符

# 只支持数字运算
[ $a -eq $b ] # 相等
[ $a -ne $b ] # 不等
[ $a -gt $b ] # 大于
[ $a -lt $b ] # 小于
[ $a -ge $b ] # 大于等于
[ $a -le $b ] # 小于等于
[ ! false ] # 非
[ $a -lt 20 -o $b -gt 100 ] # 或
[ $a -lt 20 -a $b -gt 100 ] # 与

# 逻辑运算
&& # 与
|| # 或

# 字符运算
[ $a = $b ] # 相等
[ $a != $b ] # 不等
[ -z $a ] # 长度为0，为0 true
[ -n $a ] # 长度为0，不为0 true
[ $a ] # 不空为true

# 文件测试
[ -b $file ] # 是块设备文件
[ -c $file ] # 是字符设备文件
[ -d $file ] # 是目录
[ -f $file ] # 是普通文件
[ -g $file ] # 设置了SGID
[ -p $file ] # 设置了粘着位
[ -u $file ] # 是有名管道
[ -w $file ] # 设置了SUID
[ -x $file ] # 可执行
[ -s $file ] # 是非空文件
[ -e $file ] # 文件存在

重定向

流

0 # stdin 默认键盘
1 # stdout 默认屏幕
2 # stderr 默认屏幕

重定向表示

|表示将上个命令的stdout作为stdin

> # 重定向输出 与 1>一致
&> # 重定向输出和错误输出
&>> # 附加形式重定向输出和错误输出
< # 重定向输入
2>&1  # 标准错误输出重定向到标准输出.
&>file  # 把标准输出和标准错误输出都重定向到文件file中

1>&2正确返回值传递给2输出通道 &2表示2输出通道如果此处错写成 1>2, 就表示把1输出重定向到文件2中

2>&1错误返回值传递给1输出通道, 同样&1表示1输出通道
